# 2026年2月11日 刘涛老师加餐 —— 板块分析与ETF推荐评分

## 一、文章核心观点总结

### 1. 大势判断：持股过节

- **情绪面**：市场经历1月结构性调整后，情绪已从冰点修复，当前处于"不冷不热"的健康状态
- **日历效应**：春节到两会期间，小盘股上涨概率100%，中盘股93%，大盘股60%（近15年数据）；二月份整体上涨概率86%
- **资金面**：节前融资资金流出约1000亿，节后将大比例回补，形成增量推动力
- **外围环境**：全球风险偏好上升（日本、韩国创新高，美股道指刷新记录），人民币汇率升值至6.9，外资加仓意愿强

### 2. 看好的核心板块及逻辑

文章明确提出2026年的**两大主线**：

**主线一：科技软硬件（AI全产业链）**
- AI应用软件（APP推广大战已开启，百模争霸进入商业化元年）
- 存储芯片（三大厂向HBM转产，成熟制程缺口导致涨价，至少延续到2027年）
- 晶圆代工（台积电退出成熟制程，国内厂满产接替，涨价确定）
- 封测（先进封装占比从8%提升至10%+，资本开支扩大到10-20%）
- 半导体设备（存储扩产+先进制程扩产双驱动）
- 光模块（800G到1.6T迭代，技术壁垒持续拉大）

**主线二：涨价逻辑（周期品）**
- 有色金属（铜、银、金历史新高，全球货币扩张推动）
- 化工（供给侧反内卷+原材料涨价传导）
- 煤炭（反内卷导致供给收缩，2026年价格中枢上移，高股息防御属性）

**辅助方向：**
- AI应用细分（AI营销率先落地，AI视频/IP制作轮动）
- 机器人（春晚催化，2026年量产元年）
- 电池上游（正极、负极、电解液涨价确定性高，中游需等待）

## 二、板块逻辑深度梳理

### A. 存储芯片 —— 涨价确定性最强

| 维度 | 分析 |
|------|------|
| **供给端** | 三星/SK海力士/美光向HBM转产，DDR3/DDR4成熟制程减产；三大厂扩产不急，定价权回归厂商 |
| **需求端** | AI数据中心爆发式增长，服务器DRAM需求持续超预期；手机/PC内存同步涨价 |
| **涨价幅度** | 2026Q1服务器DRAM合约价较Q4涨60-70%，整体存储合约价涨30-40% |
| **持续时间** | 缺口至少延续到2027年，国内长鑫/长存扩产填补需时间 |
| **交易窗口** | 2026年上半年确定性最强，下半年随国内扩产落地可能边际减弱 |

### B. 晶圆代工 —— 国产替代加速

| 维度 | 分析 |
|------|------|
| **核心变量** | 台积电退出成熟制程（8英寸减投），国内中芯/华虹/晶合填补缺口 |
| **产能利用率** | 国内厂满产状态，短期无过剩风险 |
| **涨价逻辑** | 供给减少+成本上升（铜、银等原材料），被动涨价确定 |
| **先进制程** | 中芯国际受限于设备，每月扩产仅4万片，节奏缓慢 |

### C. AI应用 —— 商业化元年

| 维度 | 分析 |
|------|------|
| **核心催化** | 元宝/千问/豆包/阿福春节红包大战，APP推广进入白热化 |
| **商业化标志** | 从"模型训练期"进入"用户抢占期"，大模型能力满足日常需求 |
| **细分轮动** | AI营销（率先落地）→ AI视频/IP（Sora V4催化）→ AI Agent |
| **景气验证** | 算力租赁公司业绩爆发（应用端需求拉动算力需求） |

### D. 有色金属 —— 全球货币扩张受益

| 维度 | 分析 |
|------|------|
| **宏观驱动** | 日本众议院过半→货币扩张→日债收益率新高→通胀3%+；美联储降息周期延续 |
| **价格趋势** | 伦铜突破13000美元/吨，黄金突破5600美元/盎司，白银突破110美元/盎司 |
| **A股传导** | PPI环比上涨0.4%，原材料涨价确定性极强 |

### E. 煤炭 —— 反内卷+周期复苏

| 维度 | 分析 |
|------|------|
| **供给逻辑** | 政策"反内卷"引导产能收缩，2026年煤炭供给将逐渐减少 |
| **价格中枢** | 2026年煤炭价格中枢较2025年有所提升 |
| **防御属性** | 中证煤炭指数股息率5.82%，市净率仅1.59倍 |
| **交易建议** | 低位配置，持有半年到一年，不追高 |

### F. 封测/半导体设备 —— 确定性较强的配套受益

| 维度 | 分析 |
|------|------|
| **封测** | 先进封装重要性提升（台积电资本开支占比从<10%→10-20%），国内封测厂回调即机会 |
| **设备** | 存储扩产（长鑫上海新厂2-3倍产能）+国产替代（设备适配率提升），景气度高 |

### G. 文章明确不看好的方向

| 板块 | 原因 |
|------|------|
| **功率半导体/模拟芯片** | 涨价仅为成本被动传导，非需求驱动；AI占比低，股价偏弱 |
| **CPU** | 国内无涨价空间，性能不如海外需低价竞争 |
| **MCU/SoC** | 涨价对冲存储成本上升，利润不增反可能抑制需求 |
| **商业航天** | 调整周期长（2-4个月），无近期应用场景，利好不涨 |
| **低空经济** | 有题材无资金共振，需等待新催化 |
| **光伏** | 产能过剩未解决，2026年新建规模缩减 |

## 六、风险提示

1. **外部风险**：中美关系若出现重大变化，将影响全球风险偏好和A股走势
2. **流动性风险**：节后融资回补若不及预期，短期可能有波动
3. **估值风险**：部分芯片股经过前期上涨，短期追高风险存在
4. **周期风险**：有色/化工/煤炭受全球经济周期影响，需关注PMI等先行指标
5. **政策风险**：机器人、商业航天等板块受政策调控影响较大
