# Color Block - Mac 休闲游戏设计文档

## 游戏概述
**游戏名称**: Color Block (色彩方块)

**类型**: 益智消除类休闲游戏

**平台**: Mac (基于Web技术，可在任何现代浏览器中运行)

**目标玩家**: 所有年龄段的休闲游戏玩家

## 游戏概念

Color Block 是一款简洁优雅的消除类益智游戏，玩家需要点击相邻的同色方块进行消除，获得分数。游戏设计遵循Mac的设计哲学：简洁、直观、优雅。

## 核心玩法

### 游戏规则
1. **游戏面板**: 8x8 的方块网格
2. **颜色系统**: 5种优雅的渐变色彩（符合Mac设计美学）
3. **消除机制**: 点击任意方块，所有相邻的同色方块将被消除
4. **重力系统**: 消除后，上方的方块会下落填补空缺
5. **填充机制**: 顶部会生成新的随机色彩方块
6. **得分系统**:
   - 消除的方块数量越多，分数越高
   - 计算公式: 分数 = 方块数量² × 10
   - 连续消除有额外加成

### 胜负条件
- **目标**: 获得尽可能高的分数
- **结束条件**: 无法再进行任何有效消除时游戏结束

## 视觉设计

### 设计风格
- **Mac风格**: 简洁、现代、优雅
- **色彩方案**:
  - 柔和的渐变色彩
  - 舒适的背景（浅灰色调）
  - 清晰的对比度
- **动画效果**:
  - 平滑的方块下落动画
  - 优雅的消除特效
  - 分数增加的动态反馈

### UI元素
1. **游戏标题**: 清晰的游戏名称
2. **分数显示**: 实时更新的当前分数
3. **最高分**: 本地保存的历史最高分
4. **重新开始按钮**: 随时可以重置游戏
5. **游戏说明**: 简洁的玩法提示

## 技术实现

### 技术栈
- **HTML5**: 页面结构
- **CSS3**: 样式和动画
- **JavaScript**: 游戏逻辑
- **Canvas API**: 游戏渲染

### 核心系统
1. **渲染系统**: Canvas 2D渲染
2. **输入系统**: 鼠标点击检测
3. **游戏逻辑**:
   - 方块生成算法
   - 相邻检测（洪水填充算法）
   - 重力和填充系统
4. **数据持久化**: LocalStorage保存最高分

## 游戏特色

1. **简单易学**: 规则简单，上手快
