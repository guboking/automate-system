## 🎮 游戏简介

**Color Block** 是一款专为Mac用户设计的休闲益智游戏。游戏采用简洁优雅的设计风格，玩家需要点击相邻的同色方块进行消除，追求更高的分数。游戏无需安装，可直接在浏览器中运行。

### ✨ 游戏特色

- 🎯 **简单易学**: 规则简单直观，上手即玩
- 🎨 **Mac美学**: 符合Mac系统的设计哲学，优雅简洁
- 💫 **流畅动画**: 平滑的方块下落和消除效果
- 🏆 **挑战性**: 简单规则下蕴含策略深度
- 💾 **自动保存**: 最高分自动保存到本地
- 🌐 **跨平台**: 支持所有现代浏览器

## 🎯 游戏玩法

### 基本规则

1. **游戏面板**: 8x8的彩色方块网格
2. **消除机制**: 点击任意方块，所有与之相邻的同色方块将被消除
3. **重力系统**: 消除后上方的方块会自动下落
4. **自动填充**: 顶部会生成新的随机颜色方块填补空缺
5. **游戏结束**: 当没有可消除的相邻同色方块时游戏结束

### 得分规则

- 💎 消除方块数量越多，得分越高
- 📊 计算公式: **分数 = 方块数量² × 10**
  - 消除2个方块: 40分
  - 消除3个方块: 90分
  - 消除4个方块: 160分
  - 消除5个方块: 250分
  - 以此类推...

### 游戏策略

- 🎯 优先寻找大片相连的同色方块
- 🧠 提前规划消除顺序，创造更大的连接
- 💡 避免消除小范围方块，保留机会创造大连击
- 🎲 观察整个棋盘，寻找最佳消除时机

## 🚀 如何开始

### 方法一：直接打开（推荐）

1. 在Finder中找到 `game.html` 文件
2. 双击文件，即可在默认浏览器中打开游戏
3. 开始游戏！

### 方法二：使用浏览器打开

1. 打开Safari、Chrome或其他现代浏览器
2. 将 `game.html` 文件拖拽到浏览器窗口
3. 开始游戏！

### 方法三：本地服务器（适用于开发）

```bash
# 使用Python启动本地服务器
python3 -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000/game.html
```

## 🎯 游戏技巧

1. **耐心观察**: 不要急于点击，先观察整个棋盘
2. **战略思考**: 考虑消除后的连锁反应
3. **追求大片**: 优先消除大片连接的方块
4. **保持平衡**: 避免某一区域方块过于集中
