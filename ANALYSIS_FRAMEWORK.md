# 投资分析思维框架

## 核心原则

> **角度决定难度，找到根本问题，答案自然浮现。**

---

## 一、分析前先问三个问题

### 1. 根本目的是什么？

| 表面问题 | 根本问题 |
|----------|----------|
| 这个品种会涨还是跌？ | 我为什么要交易这个品种？ |
| 能涨到多少？ | 我的目标是什么？ |
| 怎么解套？ | 怎么最快把钱赚回来？ |

**关键**：先确认目的地对不对，再谈怎么走。甚至确认"走"这个动作是否必要。

### 2. 这个品种的根本定位是什么？

- 在国家战略中处于什么位置？
- 谁在主导这个市场？（政策？市场？国际？）
- 政府/主力希望它涨还是跌？

### 3. 当前全球/宏观格局对它有什么影响？

- 地缘政治
- 货币政策（美联储、央行）
- 产业政策
- 贸易关系

---

## 二、分析层级（由高到低）

```
层级1：世界格局 / 地缘政治    ← 决定游戏规则
    ↓
层级2：国家战略 / 政策意图    ← 决定资源配置方向
    ↓
层级3：行业周期 / 供需结构    ← 决定价格中枢
    ↓
层级4：技术面 / 资金流向      ← 决定短期波动
```

**原则**：
- 上层决定下层的边界
- 下层决定上层的实现路径
- 先从高处看全貌，再落到细节

---

## 三、常见错误 vs 正确思路

| 错误思路 | 正确思路 |
|----------|----------|
| 一上来就看K线、技术指标 | 先问这个品种的根本逻辑是什么 |
| 困在供需数据里找答案 | 先问政策/周期站在哪边 |
| 死守一个品种等解套 | 问有没有更高效的路径达到目的 |
| 问"能涨到多少" | 问"谁会让它涨/跌" |
| 追求数据全面 | 追求角度精准 |

---

## 四、类比：数学解题思路

```
表面问题：怎么解这道题？
    ↓
常规思路：套公式硬算
    ↓
正确思路：
    1. 先看清题目是什么类型
    2. 选对方法/定理
    3. 再动手计算

更高维度：
    - 把代数问题转成几何图形，交点自己出来
    - 换个维度，让答案自己显现
```

**投资同理**：
- 不要在错误的角度下努力计算
- 换个角度，问题可能变简单，甚至消失

---

## 五、"起点到终点"思维

```
表面问题：怎么从 A 到 B？
    ↓
常规思路：走路？开车？飞机？哪条路线？
    ↓
升维思考：能不能直接穿越？
    ↓
更本质：为什么要去 B？
    ↓
发现：目的是传递信息 → 打电话就行
    ↓
结论：根本不需要"去"
```

**投资同理**：
- 解套的本质不是"这笔回本"，是"总资产回到目标"
- 路径可以换，不要被单一持仓困住

## 六、品种分析模板

### 第一步：定位（最重要）

| 问题 | 回答 |
|------|------|
| 这个品种在国家/全球战略中的定位？ | |
| 谁在主导这个市场？ | |
| 政策/主力的意图是什么？ | |
| 价格的"锚"在哪里？（成本线？政策目标？） | |

### 第二步：格局

| 问题 | 回答 |
|------|------|
| 当前处于什么周期位置？ | |
| 时间站在多头还是空头那边？ | |
| 什么事件会触发趋势改变？ | |

### 第三步：供需（验证，非发现）

| 问题 | 回答 |
|------|------|
| 供应端核心矛盾？ | |
| 需求端核心驱动？ | |
| 供需缺口/平衡预期？ | |

### 第四步：技术/资金（择时）

| 问题 | 回答 |
|------|------|
| 关键支撑/阻力位？ | |
| 资金流向信号？ | |
| 入场/出场时机？ | |

## 七、案例对比

### 生猪期货

| 层级 | 分析 |
|------|------|
| 国家战略 | 粮食安全，猪肉自给率95%是底线 |
| 政策意图 | 要稳，不要暴涨暴跌，主动调控产能 |
| 周期位置 | 下行末期，2026年下半年或见拐点 |
| 价格锚 | 养殖成本12000-12500是硬底 |

**根本逻辑**：政府会托底，产能去化后供需改善

### 铜期货

| 层级 | 分析 |
|------|------|
| 全球定位 | 新能源战略金属，全球定价 |
| 宏观驱动 | 美联储降息周期，弱美元支撑 |
| 供需格局 | 矿端投资不足，缺口扩大 |
| 需求驱动 | 新能源转型，刚性增长 |

**根本逻辑**：供应缺口+新能源需求+美元周期，三重支撑
