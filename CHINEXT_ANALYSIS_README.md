# 创业板指数技术分析系统

## 📊 项目简介

这是一个使用 Python 和 Playwright 开发的创业板指数（ChiNext Index, 399006）技术分析工具。系统能够抓取实时数据，计算多种技术指标，并生成详细的分析报告。

## ✨ 功能特色

- 🔍 **数据抓取**: 使用 Playwright 自动化抓取创业板实时和历史数据
- 📈 **技术指标**: 计算 MA、MACD、RSI、KDJ、布林带等多种技术指标
- 📊 **可视化报告**: 生成精美的 HTML 可视化分析报告
- 💾 **数据导出**: 支持导出 CSV 和 JSON 格式的数据
- 🤖 **自动化分析**: 自动进行技术面分析并给出操作建议

## 📁 项目文件

### 核心脚本

1. **fetch_chinext_data.py** - Playwright 版本数据抓取脚本
   - 使用浏览器自动化技术抓取数据
   - 适用于需要完整浏览器环境的场景

2. **chinext_analyzer.py** - HTTP 请求版本（推荐）
   - 直接通过 API 获取数据，更快速稳定
   - 无需浏览器环境

3. **chinext_demo.py** - 演示版本
   - 使用模拟数据展示完整分析功能
   - 适合测试和演示

### 输出文件

- **chinext_data.csv** - 原始K线数据
- **chinext_analysis.csv** - 包含所有技术指标的完整数据
- **chinext_report.json** - JSON格式分析报告
- **chinext_report.html** - 可视化HTML分析报告

## 🚀 快速开始

### 1. 安装依赖

```bash
pip3 install pandas numpy requests playwright
```

如果使用 Playwright 版本，还需要安装浏览器：

```bash
python3 -m playwright install chromium
```

### 2. 运行分析

#### 方式一：使用演示版本（推荐新手）

```bash
python3 chinext_demo.py
```

#### 方式二：使用真实数据（需要网络）

```bash
python3 chinext_analyzer.py
```

#### 方式三：使用 Playwright 版本

```bash
python3 fetch_chinext_data.py
```

### 3. 查看报告

运行完成后，在浏览器中打开 `chinext_report.html` 查看可视化分析报告。

## 📊 技术指标说明

### 1. 移动平均线 (MA)
- **MA5**: 5日均线，短期趋势
- **MA10**: 10日均线
- **MA20**: 20日均线，月线
- **MA30**: 30日均线
- **MA60**: 60日均线，季线

**判断规则**:
- 多头排列：短期均线在上，长期均线在下，看涨
- 空头排列：短期均线在下，长期均线在上，看跌
- 均线缠绕：震荡整理

### 2. MACD 指标
- **MACD线**: DIF，快速EMA - 慢速EMA
- **Signal线**: DEA，MACD的9日EMA
- **柱状图**: MACD - Signal

**判断规则**:
- 金叉：MACD上穿Signal，买入信号
- 死叉：MACD下穿Signal，卖出信号
- 柱状图增强：趋势加强

### 3. RSI 相对强弱指标
- **范围**: 0-100
- **超买区**: >70，可能回调
- **超卖区**: <30，可能反弹
- **强势区**: 50-70
- **弱势区**: 30-50

### 4. KDJ 随机指标
- **K值**: 快速线
- **D值**: 慢速线
- **J值**: 3K - 2D

**判断规则**:
- K上穿D：金叉，买入
- K下穿D：死叉，卖出
- J>100：超买
- J<0：超卖

### 5. 布林带 (Bollinger Bands)
- **上轨**: 中轨 + 2倍标准差
- **中轨**: 20日移动平均
- **下轨**: 中轨 - 2倍标准差

**判断规则**:
- 突破上轨：强势超买
- 跌破下轨：超卖反弹
- 中轨附近：震荡整理

## 📈 综合评分系统

系统会根据各项技术指标给出综合评分：

- **>3分**: 强势多头 🚀
- **1-3分**: 偏多格局 📈
- **-1到1分**: 震荡整理 ↔️
- **-3到-1分**: 偏空格局 📉
- **<-3分**: 弱势空头 🔻

## 🔧 常见问题

### Q1: 数据获取失败怎么办？
A: 可能是网络问题或API限制，建议：
- 检查网络连接
- 稍后重试
- 使用演示版本（chinext_demo.py）

### Q2: Playwright 安装失败？
A: 建议直接使用 chinext_analyzer.py，无需 Playwright

### Q3: 如何自定义分析参数？
A: 在脚本中修改以下参数：
- MA周期：`periods=[5, 10, 20, 30, 60]`
- MACD参数：`fast=12, slow=26, signal=9`
- RSI周期：`period=14`
- 布林带：`period=20, std_dev=2`

### Q4: 如何添加更多技术指标？
A: 可以在 `ChiNextAnalyzer` 类中添加新的计算方法，例如：
- ATR（真实波动幅度）
- OBV（能量潮）
- WR（威廉指标）
- CCI（顺势指标）
