# 补剂评估框架 Skill

## 触发条件

当用户提到以下关键词时启动：
- 「这个补剂怎么样」「XXX 能吃吗」「XXX 有用吗」
- 「中医角度」「归经」「成分分析」
- 「副作用」「相互作用」「禁忌」

## 证据等级说明

| 等级 | 定义 |
|------|------|
| A级 | 多项RCT研究支持，Meta分析确认 |
| B级 | 单项RCT或多项观察性研究 |
| C级 | 小样本研究或体外/动物实验 |
| D级 | 传统使用经验，缺乏现代研究 |

---

## 常见交互作用数据库

### CYP450 抑制剂
- 西柚汁、姜黄素、黑胡椒碱（CYP3A4）
- 绿茶提取物（CYP2C9）

### CYP450 诱导剂
- 圣约翰草（多个CYP）
- 大蒜提取物（CYP2E1）

### 常见配伍禁忌
- 铁 + 钙 → 相互影响吸收
- 维生素K + 抗凝药 → 拮抗作用
- 人参 + 咖啡因 → 协同增强
