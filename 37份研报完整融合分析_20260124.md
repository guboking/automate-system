## 二、顶层框架（所有研报一致）

### 2.1 康波周期定位

```
当前位置：康波萧条期（2018-2028）
         ↓
破局路径：AI技术革命
         ↓
博弈形式：新冷战（中美对抗）
         ↓
投资主线：资源（防守）+ 科技（进攻）
```

**研报原话印证**：
- 11.10买方："从2018年开始，我们进入了康波萧条期"
- 1.12买方："新冷战格局下，自主可控资产价值极其巨大"
- 12.29买方："科技是主旋律，消费不可能把中国变成领先国"
- 1.23特邀："35个卡脖子项目基本解决，中国获得供应链主导权"

### 2.2 康波周期详解

| 阶段 | 特征 | 时长 |
|------|------|------|
| 复苏期 | 新技术展开 | 15年 |
| 扩张期 | 技术快速渗透 | 15-20年 |
| 衰退期 | 技术红利递减 | 4-5年 |
| **萧条期（当前）** | 技术停滞，贫富差距加剧 | 10-15年 |

**萧条期特征**：
1. 贫富差距极度加剧
2. 逆全球化加速
3. 通缩压力
4. 政治极化

**破局路径**：
- AI成功 → 带动全球走出萧条，进入复苏期，牛市延续
- AI失败 → 牛市终结，历史上的出路是战争

### 2.3 新冷战格局

**定义**：全球进入以中美对抗为主轴的新冷战格局

**具体表现**：
| 领域 | 事件 |
|------|------|
| 科技封锁 | 阻止阿斯麦向中国销售光刻设备 |
| 数据主权 | TikTok出售要求 |
| 技术管控 | 星链禁止传播到中国 |
| 稀土反制 | 中国限制稀土出口 |

**投资逻辑**：一切自主可控的资产都是对抗性资产

### 2.4 供应链主导权与趋势性慢牛（新增）

**1.23特邀课程核心观点**：

```
35个卡脖子项目基本解决
         ↓
中国获得供应链主导权
         ↓
货币发行能力增强（货币政策历史性转向）
         ↓
流动性从周期性变为趋势性
         ↓
趋势性慢牛（类似美股2008年后）
```

**关键变化**：
- 中国已具备供应链主导权，不再担心"掀桌子"
- 货币政策从"适度宽松"转向"适度紧缩"的逆向操作空间打开
- A股可能复制美股2008年后的十年长牛

### 2.5 国家队控盘慢牛（新增）

**1.21行业加餐核心观点**：

```
低利率环境
    ↓
资产荒（固收类资产收益率下行）
    ↓
资金流向：黄金 + 股市
    ↓
股市中国家指定方向 = 新质生产力
```

**国家意图明确**：通过控盘实现慢牛，避免暴涨暴跌

### 2.6 市场性质判断

| 维度 | 判断 | 数据支撑 |
|------|------|----------|
| **市场定性** | 十年一遇大牛市 | 对标06-07、14-15 |
| **当前阶段** | 主升浪加速期 | 季线金叉后第三个季度 |
| **大顶信号** | 远未到 | 成交量需5万亿+，开户600万+/月 |
| **指数空间** | 4400-5200点 | 沪深300 PE 14倍 vs 历史高点19-45倍 |
| **市场特征** | 趋势性慢牛 | 供应链主导权支撑流动性 |

## 九、核心结论

### 一句话总结

> **康波萧条期+新冷战格局下，中国已获得供应链主导权，趋势性慢牛确立。资源是防守底仓，AI科技是进攻主线。商业航天是本周最强共识，小核酸/脑机接口是新增亮点。注意春节附近可能见阶段高点。**

### 逻辑闭环图

```
康波萧条期（2018-2028）
       ↓
内部矛盾外化 → 新冷战格局
       ↓
35个卡脖子项目基本解决 → 供应链主导权确立
       ↓
货币政策历史性转向 → 流动性从周期性变为趋势性
       ↓
两大投资主线：
├── 防守端：资源（对抗货币超发、萧条期核心配置、资产荒首选）
│   └── 黄金 > 铜 > 稀土 > 白银
│
└── 进攻端：AI（押注复苏期、自主可控）
    └── 商业航天 > 半导体设备 > 存储 > 机器人 > 小核酸/脑机接口
```

### 操作纪律

1. **不追高**：牛市追涨容易亏钱
2. **保持底仓**：ETF方式参与硬科技
3. **分歧是买点**：高位股调整时加仓
4. **三个目标**：不犯大错、及格、跟上主线
5. **大妈票可买**：有业绩支撑的蓝筹
6. **妖股要避**：纯题材炒作风险大
7. **注意春节节奏**：历史上12月启动的行情春节附近见顶概率高
