## 二、COMEX库存与持仓结构

### 2.1 COMEX白银库存
- 2020年：3.46亿盎司
- 2025年中：注册仓单降至约8,200万盎司（**-75%**）
- 2025年上半年：银行将9,500万盎司从"注册"转为"合格"类别，锁定供应
- 市场处于**现货升水（backwardation）**状态，借贷利率一度突破100%

### 2.2 CFTC COT持仓（2026/1/27）
- 管理基金净多头：~118,159张合约（100盎司合约），持仓偏高
- 商业空头（5,000盎司合约）：净空头~44,056张
- 未平仓合约：488,463张（100盎司），为历史高位

**推论：** 投机性多头拥挤，但物理市场紧张支撑。库存5年消耗75%是结构性事实，非短期投机能解释。

## 四、货币属性与宏观定价

### 4.1 美联储利率路径
- 当前联邦基金利率：3.50%–3.75%（2025年底已降175bp）
- 2026年预期：再降50–100bp至3.00%–3.25%
- Powell 2026年5月卸任，新主席人选Kevin Warsh偏鹰派（偏好缩表）

### 4.2 实际利率
- 核心PCE：~2.8%（仍高于2%目标）
- 10年期国债收益率目标：~3.75%
- 隐含实际收益率：~1.0–1.5%（从高位回落趋势）
- **历史规律：实际利率下降 → 白银上涨**，当前环境中性偏正面

### 4.3 美元指数（DXY）
- 当前：~98（多月低点）
- 2026年预测：94–100区间震荡，趋势性走弱
- BRICS去美元化加速：美元储备占比降至58.2%（1995年以来最低）

### 4.4 M2货币供应
- 美国M2：$22.4万亿，同比+4.6%（温和扩张）
- 全球M2：突破$123.3万亿历史新高
- 货币扩张环境总体利好贵金属

### 4.5 央行购金溢出效应
- 2025年央行购金：328吨（IMF数据）/ 863吨（WGC估算）
- 2026年预测：755–800吨
- 央行持续增配黄金 → 推高金价 → 压缩金银比 → 溢出利好白银

**宏观面定价推论：**
- 降息周期 + 弱美元 + 货币扩张 = 贵金属友好环境
- 但Warsh鹰派风险可能限制上行
- **宏观面合理区间：$55–85**

## 五、成本底线分析

### 5.1 AISC数据

| 矿企 | 2025年AISC（$/oz） |
|-------|-------------------|
| Pan American Silver | $14.50–16.00 |
| Juanicipio（世界最低成本） | $6.00–8.00 |
| Hecla Mining | ~$11.00 |
| First Majestic Silver | $19.89–21.27 |
| **行业均值** | **$15–20** |
| **行业H1 2025均值** | **$13.0** |

### 5.2 成本曲线逻辑

- 行业平均AISC：$15–20/oz
- 90分位成本估计：$22–26/oz（基于历史趋势外推，2024年AISC中位数$26.86的报告数据）
- **副产银的特殊逻辑：** 当铜价/锌价下跌时，铜矿/锌矿减产 → 副产银供给下降 → 反而推高银价

### 5.3 成本底价推论

- **绝对底部：$22–27/oz**（90分位矿山AISC，低于此价大量矿山关停）
- **可持续生产价格：$30–40/oz**（覆盖扩产资本开支+合理回报）
- 当前$70远高于成本底部，说明**投资溢价/稀缺溢价**才是主要定价因素

## 六、历史类比与估值锚

### 6.1 金银比回归分析

| 假设金银比 | 当前金价$4,815时的白银价格 |
|-----------|--------------------------|
| 当前 55:1 | $87.5（实际值） |
| 5年均值 80:1 | $60.2 |
| 历史均值 65–70:1 | $68.8 – $74.1 |
| 长期均值 54:1 | $89.2 |

**如果金价稳定在$4,800附近，金银比在55–70之间意味着白银$68–$89。**

### 6.2 通胀调整后的历史高点

| 历史事件 | 名义价格 | 2025/2026年等价 |
|---------|---------|----------------|
| 1980年1月峰值 | $49.45 | ~$170–185 |
| 2011年4月峰值 | $49.20 | ~$71（已被超越） |

- 当前$70已超过2011年峰值的通胀调整价
- 但距1980年通胀调整价仍有~60%的差距
- 1980年是极端投机事件（Hunt兄弟逼仓），不宜作为合理估值参考

### 6.3 白银在全球金融资产中的占比
- 白银市场规模极小（全部地上存银约16亿盎司 × $70 ≈ $1,120亿）
- 仅占全球金融资产（$500万亿+）的0.02%
- 极小的资金流入即可产生巨大价格影响

## 八、综合定价推导

### 各维度价格区间汇总

| 定价维度 | 底部 | 中枢 | 上限 | 权重 |
|---------|------|------|------|------|
| 供需基本面 | $45 | $65 | $75 | 30% |
| 宏观/货币 | $55 | $70 | $85 | 20% |
| 成本底线 | $27 | $35 | $40 | 10% |
| 金银比回归 | $60 | $74 | $89 | 25% |
| 资金流向/动量 | $55 | $75 | $100+ | 15% |

### 加权合理价格区间

**核心合理区间：$55 – $80/oz**

- **下限 $55：** 金银比回归至80:1 + 供需缺口收窄 + 投机退潮的组合
- **中枢 $68：** 供需缺口持续 + 温和宏观利好 + 金银比维持65:1
- **上限 $80：** 金银比保持55:1 + ETF持续流入 + 降息加速

### 极端情景

| 情景 | 价格 | 概率评估 |
|------|------|---------|
| 供需危机 + 挤空 | $100–150 | 15% |
| 基本面支撑 + 正常投机 | $55–80 | 50% |
| 衰退 + 工业需求崩塌 | $35–55 | 25% |
| 美元暴涨 + 通缩 | $25–35 | 10% |

## 九、质疑与反例——可能使上述区间失效的情景

### 情景1：光伏去银化加速超预期
- **触发条件：** 铜浆技术在TOPCon路线上突破可靠性瓶颈，2027年渗透率超30%
- **影响：** 光伏银需求从200Moz降至100Moz，年供需缺口消失甚至转为盈余
- **价格影响：** 可能将合理价格拉低至$35–45
- **当前评估：** 技术上正在加速（LONGi、Aiko已GW级生产），但TOPCon主流路线的铜氧化问题尚未解决，2-3年内难以大规模替代。**中期风险，非近期威胁。**

### 情景2：全球经济深度衰退
- **触发条件：** 贸易战升级 + 美联储被迫加息 + 信贷危机
- **影响：** 工业需求（占总需求55%+）骤降10-15%，ETF被赎回
- **价格影响：** 可能跌至$30–40（参考2020年3月短暂跌至$12）
- **当前评估：** 美国经济目前仍韧性较强，但Warsh鹰派倾向 + 贸易政策不确定性构成风险。**尾部风险。**

### 情景3：大规模库存释放或新矿投产
- **触发条件：** 中国/俄罗斯释放战略储备；多个大型银矿同时投产
- **影响：** 年供给增加5-10%，缺口快速收窄
- **价格影响：** 合理中枢下移至$45–55
- **当前评估：** 新矿开发周期7-10年，短期无大型项目即将投产。库存释放属于政策黑天鹅。**低概率。**

### 情景4：美元异常走强
- **触发条件：** 全球避险 → 美元回到DXY 110+ → 实际利率飙升
- **影响：** 贵金属全面承压，金银比可能重新走阔至80-90
- **价格影响：** 白银可能回落至$45–55
- **当前评估：** 结构性去美元化趋势限制了DXY大幅走强的空间，但短期避险事件可能造成脉冲式冲击。**中等概率。**

### 情景5：投机泡沫破裂（当前最相关风险）
- **触发条件：** 1月冲$121后已开始。管理基金多头过度拥挤，获利盘涌出
- **影响：** 类似2011年从$49跌至$30的剧烈去杠杆
- **价格影响：** 短期可能超跌至$45–55，然后在基本面支撑下回升
- **当前评估：** **正在发生。** 从$121到$70已跌40%+。关键问题是底部在哪里。

## 十、结论

### 独立推导的白银合理价格区间

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  成本绝对底部        合理区间           投机区间  │
│  ├──────┤    ├────────────────┤   ├──────────┤  │
│  $25    $40  $55            $80   $100    $150  │
│               ▲                                 │
│          核心中枢 $65–70                         │
│                                                 │
│  当前价格 ~$70 ← 处于合理区间上半部              │
│                                                 │
└─────────────────────────────────────────────────┘
```

| 结论项 | 判断 |
|--------|------|
| **合理价格区间** | **$55 – $80/oz** |
| **核心中枢** | **$65 – $70/oz** |
| **成本支撑底部** | $25 – $40/oz |
| **投机峰值潜力** | $100 – $150/oz（需极端条件） |
| **当前$70的位置** | 合理区间上半部，但已从泡沫区回归 |

### 核心逻辑链

1. **成本底部$25-40**给出了绝对下限（矿山关停价）
2. **连续5年结构性缺口**将均衡价格推高至远超成本的$55+
3. **金银比55-70:1 × 金价$4,800**锚定$68-87的参考区间
4. **宏观环境**（降息+弱美元+M2扩张）提供$55-85的支撑
5. **加权后的核心区间$55-80**反映了基本面与投机因素的平衡

### 当前$70的定性判断

**合理但不便宜。** 从$121回调至$70已释放了相当部分的投机溢价，当前价格处于多维度推导的合理区间内。但考虑到：
- 管理基金多头仍然拥挤
- 金银比55:1已低于长期均值
- 去银化技术加速的中期风险

**短期（1-3个月）可能继续下探$55-60区域寻底，之后在$55-80区间内波动。** 若宏观环境改善（降息加速、DXY走弱），$80+可期。若衰退风险加剧，$40-50不可排除。
