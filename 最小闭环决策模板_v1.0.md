# 最小闭环决策模板 v1.0（Markdown 可复用版）

> 目标：避免情绪判断，用结构验证真假趋势。

## 0) 使用说明
- 适用场景：资产配置、趋势判断、仓位决策、风险复盘。
- 使用原则：先结构、后观点；先验证、后结论。
- 输出要求：必须给出触发条件、失效条件、止损、概率区间。

## 风险与局限（硬卡口）

- 若涉及实时数据，必须先查证。

## 决策顺序（执行清单）

1. 是不是 → 变量验证  
2. 为什么 → 因果链  
3. 稀缺性 → 是否替代  
4. 象限判断 → 资金/结构  
5. 强人推演 → 谁受益  
6. 闭环输出 → 触发 + 止损 + 概率
